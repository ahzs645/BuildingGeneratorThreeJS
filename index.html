A growing set of procedural generators, ported from Blender to Three.js by two routes: a bake pipeline (evaluate the node graph in Blender → glTF) and a from-scratch geometry-nodes VM that executes the dumped node graph directly in TypeScript — no Blender at runtime, live sliders, static-deployable.
