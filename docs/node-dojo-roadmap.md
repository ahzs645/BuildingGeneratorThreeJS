## Recommended next targets

1. **Chrome Asset Library** — 43 active objects, 26 modifier roots, 404 groups. Twenty-three root families have browser evidence, including both Pixel Marker users, both Soft Pixel Marker roots, Text Soup, Blunt Outline Crayon, Blunt Metal Marker, and the 3D Chrome Crayon Generator, the exact non-surface Mesh to Curve helper, Chain and Mace surface parity, the near-exact UI Window Generator, both editable String to Text roots, and the exact authored Chrome Tools text plate; continue in increasing reachable-node complexity.
2. **N03D 3D-printing Utilities** — 73 active objects, 39 roots, 639 groups. Dowel and Bolt are complete; Clevis Pin is live with a documented screw/sweep parity gap. Continue with the next isolated print utility.
3. **New Joint Generators** — 32 active objects, 13 roots, 572 groups. Pipe and dowel generators are the clearest individual products; Bubble Putty is a larger field/volume target.
4. **More Math Clay surfaces** — 33 active objects and 14 root families. Low-risk as baked gallery pieces; substantially more work for a live TPMS evaluator.
5. **The Nodes Node** — 17 active objects, 12 roots, 5,498 total nodes. Treat as a collection, not one graph.
